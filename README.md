# ServiTecnicoMaq - Servicio Técnico Integral

Landing page profesional para **ServiTecnicoMaq**, especialistas en reparación de electrodomésticos, sistemas de calefacción y climatización personalizada en la Región del Biobío, Chile.

## 🚀 Sitio Web
Puedes ver el despliegue en vivo aquí:
**[https://nanculefservice.github.io/servitecnicomaq/](https://nanculefservice.github.io/servitecnicomaq/)**

## 🛠️ Sobre el Proyecto
Este sitio fue diseñado para facilitar el diagnóstico remoto y la gestión de visitas técnicas, permitiendo a los usuarios enviar evidencias multimedia (fotos y videos) de sus averías directamente por WhatsApp.

### Características Principales:
- **Diseño Responsivo:** Navegación fluida en móviles, tablets y escritorio.
- **Formulario de Solicitud:** Sistema de solicitud de visita técnica con selección de servicios, electrodomésticos, marcas y comunas.
- **Flujo de Diagnóstico Rápido:** Sección dedicada que explica cómo capturar y enviar el problema para una evaluación inmediata.
- **Optimización de Conversión:** Botones estratégicos de contacto directo a WhatsApp y correo electrónico.

## 📋 Servicios Especializados
- **Electrodomésticos:** Reparación de lavadoras, secadoras y refrigeración (Samsung, LG, Mabe, Fensa, Whirlpool, entre otras).
- **Calefacción:** Instalación y mantenimiento de calefont a gas, calefactores eléctricos y termos de agua.
- **Climatización:** Aire acondicionado, ventilación industrial y servicio comercial para negocios.

## 📍 Cobertura y Ubicación
